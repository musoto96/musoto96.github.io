About this site

This website was created with the Hugo framework and the nostyleplease theme.

The images used as banners around the site, where created using AI, more specifically, Bing Image Creator powered by DALL-E.

The prompt was something like this example:

> An image with white backgound of an Aztec warrior with a skull face, using a laptop

I always had to do some cleaning up with Gimp but, for the most part, they came out pretty good straight from the Bing engine.
